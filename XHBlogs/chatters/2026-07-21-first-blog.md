## 🌱 一个新开始

折腾了好几天，这个博客终于从零到一站起来了。

一直想要一个属于自己的角落——不是为了给谁看，而是想在这个快节奏的时代里，给自己留一个可以慢下来的地方。记录学到的技术、踩过的坑、拍到的照片，以及那些转瞬即逝的想法。

## 为什么叫"数字花园"？

比起传统的博客，我更喜欢 **Digital Garden（数字花园）** 这个概念——

- 🌿 文章不是一成不变的，会随着我的成长不断更新、修剪
- 🌸 各篇文章之间像植物一样相互连接，形成知识网络
- 🪴 不在乎更新频率，只在乎每一篇是否用心

这里会有：

| 模块 | 内容 |
|------|------|
| 📝 **文章** | 技术笔记、经验总结、深度思考 |
| 💬 **说说** | 日常碎碎念，当朋友圈用 |
| 📰 **杂谈** | 数码评测、AI 工具对比、好物推荐 |
| 📸 **照片墙** | 摄影作品、生活随拍 |
| 🎵 **音乐** | 网易云歌单，写代码时的 BGM |
| 📌 **项目** | GitHub 仓库展示 |
| 👋 **关于** | 自我介绍与研究动态 |
| 🔗 **友链** | 小伙伴们的博客 |

## 技术栈

这个博客基于 [Volga](https://github.com/jaychou4399/Volga) 搭建，使用 Next.js + Tailwind CSS，部署在 Vercel 上。

从买域名到配 DNS 到第一次成功部署，踩了不少坑，但也学到了很多。最大的感受是：**动手做比看一百篇教程都有用**。

## 写在最后

不知道这个博客能坚持更新多久，但至少现在——2026 年的夏天，我有了一个属于自己的小角落。

希望一年后回头看，这里已经长成了一片小森林 🌲
