## 🌱 一个新开始

折腾了好几天，这个博客终于从零到一站起来了。

一直想要一个属于自己的角落——不是为了给谁看，而是想在这个快节奏的时代里，给自己留一个可以慢下来的地方。记录学到的技术、踩过的坑、拍到的照片，以及那些转瞬即逝的想法。

## 为什么叫"数字花园"？

比起传统的博客，我更喜欢 **Digital Garden（数字花园）** 这个概念——

- 🌿 文章不是一成不变的，会随着我的成长不断更新、修剪
- 🌸 各篇文章之间像植物一样相互连接，形成知识网络
- 🪴 不在乎更新频率，只在乎每一篇是否用心

这里会有：

| 模块 | 内容 |
|------|------|
| 📝 **文章** | 技术笔记、经验总结、深度思考 |
| 💬 **说说** | 日常碎碎念，当朋友圈用 |
| 📰 **杂谈** | 数码评测、AI 工具对比、好物推荐 |
| 📸 **照片墙** | 摄影作品，按时间和主题整理 |
| 🔧 **项目** | GitHub 仓库展示，学习进度追踪 |
| 🎵 **音乐** | 网易云歌单，边听边写 |

## 技术栈

这个博客基于 [Volga](https://github.com/jaychou4399/Volga) 搭建：

- **Next.js 15** + React 19——静态生成，速度快
- **Tailwind CSS**——原子化样式，改起来方便
- **Vercel** 部署——推送即上线，零运维
- **Markdown** 写作——用习惯的编辑器写文章

## 关于我

自动化专业在读，嵌入式方向学习中。平时鼓捣 STM32、Python，偶尔写写 Java。课余爱好摄影，喜欢记录生活中的光影。
